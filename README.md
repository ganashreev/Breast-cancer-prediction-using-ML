# Breast Cancer Detection Using Deep Learning

This project uses a VGG16-based Convolutional Neural Network (CNN) to classify breast cancer histopathology images as Benign or Malignant.

## Technologies Used
- Python
- TensorFlow
- Keras
- Streamlit
- OpenCV

## Features
- Image Upload
- Real-time Prediction
- Deep Learning Model
- User-Friendly Interface

## Accuracy
Approximately 94–95%
